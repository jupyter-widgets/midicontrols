
ipymidicontrols
=====================================

Version: |release|

A Jupyter widget for interfacing with MIDI controllers.


Quickstart
----------

To get started with ipymidicontrols, install with pip::

    pip install ipymidicontrols

or with conda::

    conda install ipymidicontrols
